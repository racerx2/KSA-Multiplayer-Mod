# KSA Multiplayer Mod

A multiplayer modification for Kitten Space Agency (KSA) that enables real-time cooperative gameplay where multiple players can see each other's spacecraft during orbital missions, maneuvers, and surface operations.

## Repository Structure

- **[KSA-Multiplayer-Mod/](KSA-Multiplayer-Mod/)** - Source code for the multiplayer mod
- **[KSA-Multiplayer-Package/](KSA-Multiplayer-Package/)** - Installer and launcher files

## Quick Start

Download the latest release from the [Releases](https://github.com/racerx2/KSA-Multiplayer-Mod/releases) page.

For detailed installation and usage instructions, see the [mod README](KSA-Multiplayer-Mod/README.md).

## Features

- Real-time vehicle synchronization between players
- Event-based architecture for efficient networking
- In-game chat system
- Player nametags above vehicles
- Time synchronization
- System configuration validation
- EVA astronaut support

## Requirements

- Kitten Space Agency v2025.12.24.3014 or compatible
- .NET 9 Runtime
